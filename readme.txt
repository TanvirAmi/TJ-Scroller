TJ Scroller is an ultra-lightweight WordPress plugin, sized at just 12 KB. It is designed for optimal performance, ensuring smooth scrolling functionality without affecting your website's page load time.


==Description==

This plugin will add a scroll to top functionality.
After installiation you can see a button on footer right.

==Installation==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `TJ Scroller` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3.You have done.
